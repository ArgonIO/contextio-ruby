Overview

Include this into an APIResource to delete an instance of it.

Implement an instance method `primary_key` that returns the calue of the primary key for the resource. For instance, the API uses the provider consumer key for the primary identifier for OAuth Providers, so OAuthProvider#primary_key returns the provider_consumer_key.

Also, implement the class method `url` that returns a string, which is the last part of the list url. For instance, the listing url for accounts is 'api.context.io/2.0/accounts', so Account.url returns 'accounts'.
